# 25-pt-stencil

ModelZoo validation benchmark for the CSL `25-pt-stencil` example.

The PyTorch graph runs a fixed-count 3D time-marching stencil. It uses the
Minimig coefficients from the CSL source for distances 1 through 4 in each axis,
keeps two explicit wavefield tensors, multiplies by the deterministic velocity
field, and injects a deterministic Gaussian source tensor.

This is a static tensor analogue. It does not model CSL colors, entrypoints,
PE-to-PE wavelets, command-stream unblocking, timestamp buffers, or the final
host-side max/min timing transfer. The returned loss is the mean-squared error
against the deterministic PyTorch reference generated by the data processor.
